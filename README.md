## [WIP] Projeto de integração da Twitter API com React & Node.js + Typescript

Este é um protótipo de aplicativo web que estou desenvolvendo utilizando as tecnologias ReactJS, Node.js e Typescript, com o objetivo de exibir minhas habilidades de desenvolvimento em meu portfólio. O foco do aplicativo é a integração com a Twitter API V2, permitindo que os usuários pesquisem, filtrem e exibam tweets de forma dinâmica. [Demonstração online](https://getsweeter.vercel.app/)

[LinkedIn](https://www.linkedin.com/in/leosoares94/)\
[Instagram](https://instagram.com/leosoares94)

## Especificações técnicas:

### Front-end:
- Iniciado com [Vite](https://vitejs.dev/)
- Alguns componentes: [Chakra UI](https://chakra-ui.com/)
- Gerenciamento de estado: [Zustand](https://github.com/pmndrs/zustand)
- Estilo: [Styled Components](https://styled-components.com/)
- Linting: [ESLint](https://eslint.org/)
